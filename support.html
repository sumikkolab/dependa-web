不具合の報告

不具合を報告する際は、以下の情報をお知らせください。

  1. バージョン番号（ホーム画面に表示）
  2. 再現手順（どのような操作をしたか）
  3. エラーメッセージ（表示された場合）
  4. スキャン対象のエコシステム（Python / Node.js）

注意: プロジェクトのソースコードや依存ファイルをそのまま送付しないでください。不具合の再現に必要な最小限の情報のみお送りください。

プランと機能

プラン 内容
Free Python スキャン、ローカル脆弱性照合、ライセンス分析、SBOM・HTML レポート生成。
Pro Free の全機能に加え、Node.js 対応、OSV API 脆弱性照合、PyPI ライセンス補完、Delta Scan、AI プロンプト生成。

よくある質問

Q: オフラインで使えますか？

A: はい。Free 版の全機能はオフラインで動作します。オンライン機能（OSV API、PyPI API）は Pro 版で、ユーザーが明示的に有効化した場合のみ使用されます。

Q: Pro 版の購入方法は？

A: アプリ内の設定画面から Microsoft Store 経由で購入できます。

Q: CLI からも使えますか？

A: はい。Dependa.exe scan --path <path> で CLI スキャンが可能です。詳細はCLI リファレンスをご覧ください。

FAQ

Q: Can I use it offline?

A: Yes. All Free features work offline. Online features (OSV API, PyPI API) are Pro-only and only used when explicitly enabled by the user.

Q: How do I purchase Pro?

A: You can purchase through the Microsoft Store via the app's Settings screen.

Q: Can I use it from the command line?

A: Yes. Run Dependa.exe scan --path <path> for CLI scanning. See the CLI Reference for details.
