プライバシーポリシー

Dependa — SumikkoLab｜最終更新日: 2026-03-13

1. はじめに

本プライバシーポリシーは、デスクトップアプリケーション「Dependa」（以下「本アプリ」）における情報の取り扱いについて説明します。

2. 本アプリが収集しない情報

本アプリは以下の情報を収集しません。

  • 氏名、メールアドレス、電話番号等の個人情報
  • 位置情報
  • 利用状況の統計データ（テレメトリ）
  • Cookie やトラッキング情報
  • プロジェクトのソースコード

3. ローカルに保存される情報

本アプリは以下の情報をお使いの端末にローカル保存します。外部サーバーへの送信は行いません。

  • アプリ設定: 言語設定、オンライン機能の有効/無効、利用規約の同意状態
  • ライセンス状態: Free / Pro の状態
  • スキャン結果: SBOM ファイル、HTML レポート（いずれもユーザーが指定した出力先に保存）

保存先: %LocalAppData%\Dependa\settings.json

4. オンライン機能利用時の通信

本アプリにはオプションのオンライン機能があります。いずれもユーザーが明示的に有効化した場合のみ動作します。

4.1 OSV API（脆弱性照合）

  • 通信先: https://api.osv.dev/（Google が運営する OSV データベース）
  • 送信内容: パッケージ名、バージョン番号、エコシステム名（例: PyPI, npm）
  • 目的: 最新の既知脆弱性情報の照合
  • 個人情報の送信: なし

4.2 PyPI API（ライセンス情報補完）

  • 通信先: https://pypi.org/（Python Package Index）
  • 送信内容: パッケージ名
  • 目的: ライセンス情報の補完取得
  • 個人情報の送信: なし

4.3 通信の無効化

上記のオンライン機能はいずれも設定画面から無効化できます。無効時はインターネット通信を一切行いません。

5. 課金に関する情報

Pro 機能の購入は Microsoft Store の仕組みを通じて行われます。決済情報は Microsoft が管理し、本アプリの開発者が直接取得することはありません。

6. 第三者への提供

本アプリが取得または生成した情報を第三者に提供することはありません。

7. サポート時のデータ取り扱い

不具合調査のために提供されたデータは、調査目的以外に使用せず、調査完了後速やかに（原則 30 日以内）破棄します。

8. ローカル処理

Dependa は解析をローカル環境で実行し、ソースコードやプロジェクトファイルを外部サーバーへ送信することはありません。Free 機能は完全にオフラインで動作します。Pro 機能ではユーザーがオンライン照合を有効化した場合のみ、OSV などの公開脆弱性データベースを参照します。

9. 変更について

本プライバシーポリシーは必要に応じて更新される場合があります。重要な変更がある場合はアプリ内で通知します。

10. お問い合わせ

プライバシーに関するお問い合わせは以下までお願いします。

support@sumikkolab.com

Privacy Policy

Dependa — SumikkoLab | Last updated: March 13, 2026

1. Introduction

This Privacy Policy explains how information is handled in the desktop application "Dependa" (hereinafter "the App").

2. Information We Do Not Collect

The App does not collect any of the following:

  • Personal information such as names, email addresses, or phone numbers
  • Location data
  • Usage statistics or telemetry
  • Cookies or tracking information
  • Project source code

3. Information Stored Locally

The App stores the following information locally on your device. No data is sent to external servers.

  • App settings: Language preference, online feature toggle, terms acceptance status
  • License state: Free / Pro status
  • Scan results: SBOM files and HTML reports (saved to user-specified output locations)

Storage location: %LocalAppData%\Dependa\settings.json

4. Network Communication for Online Features

The App includes optional online features. All online features operate only when explicitly enabled by the user.

4.1 OSV API (Vulnerability Lookup)

  • Endpoint: https://api.osv.dev/ (OSV database operated by Google)
  • Data sent: Package name, version number, ecosystem name (e.g., PyPI, npm)
  • Purpose: Matching against the latest known vulnerability data
  • Personal information sent: None

4.2 PyPI API (License Metadata)

  • Endpoint: https://pypi.org/ (Python Package Index)
  • Data sent: Package name
  • Purpose: Supplemental license information retrieval
  • Personal information sent: None

4.3 Disabling Online Features

All online features listed above can be disabled from the Settings screen. When disabled, the App makes no internet connections whatsoever.

5. Billing Information

Pro feature purchases are processed through the Microsoft Store platform. Payment information is managed by Microsoft, and the App developer does not directly access any billing data.

6. Third-Party Sharing

The App does not share any collected or generated information with third parties.

7. Data Handling During Support

Data provided for investigation purposes will not be used for any other purpose and will be promptly destroyed (within 30 days) after investigation is complete.

8. Local Processing

Dependa performs analysis locally and does not transmit source code or project files to external servers. Free features run completely offline. Pro features optionally query public vulnerability databases such as OSV when the user enables online lookup.

9. Changes to This Policy

This Privacy Policy may be updated as needed. Users will be notified within the App of any significant changes.

10. Contact

For privacy-related inquiries, please contact:

support@sumikkolab.com
