# Contributing

English is preferred for all issues and communication. Japanese is also accepted.

This is a small, focused project. Development priorities are set internally. Pull requests are not accepted. Feature requests are not negotiated.

If you find a bug, file an issue. It gets read. Response times may be slow — this is expected.

Security issues: see [SECURITY.md](SECURITY.md).

## What makes a good bug report

- Dependa version (shown in Settings > About or `Dependa.exe version`)
- Windows version
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable
