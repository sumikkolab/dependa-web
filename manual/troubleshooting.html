v2.0 でよくある問題

SBOM ファイルがインポートできない

以下を確認してください。

  • ファイル形式が CycloneDX 1.4/1.5 JSON または SPDX 2.3 JSON であること。
  • XML 形式の SBOM は v2.0 では未対応です。JSON 形式に変換してからインポートしてください。
  • ファイルが壊れていないこと。JSON として正しくパースできるか確認してください。

ライセンスプリロードエラー

起動時に「License preload failed」が表示される場合、ライセンスデータベースの読み込みに問題が発生しています。アプリを再起動するか、設定ファイルを削除して再起動してください。

レビュー件数が 5 件で止まる

Free 版ではレビュー可能な件数が 5 件に制限されています。すべてのパッケージをレビューするには Pro ライセンスが必要です。

レポートにウォーターマークが表示される

Free 版で生成された HTML レポートには「Free Edition」のウォーターマークが表示されます。これは正常な動作です。ウォーターマークなしのレポートが必要な場合は Pro にアップグレードしてください。

設定画面 — トラブル時はここで Pro ライセンス状態やオンライン設定を確認

設定のリセット方法

設定を初期状態に戻すには、以下の手順を実行してください。

  1. Dependa を終了します。
  2. 以下のファイルを削除します。
%LocalAppData%\Dependa\settings.json

次回起動時に設定ファイルがデフォルト値で再作成されます。

設定ファイルを削除すると、利用規約の同意状態もリセットされます。再度同意が必要になります。

WebView2 Runtime が見つからない場合

Dependa は一部の UI 表示に Microsoft Edge WebView2 Runtime を使用します。以下のエラーが表示された場合はインストールが必要です。

  • Windows 11 には標準で含まれています。通常はインストール不要です。
  • Windows 10 で表示された場合は、Microsoft の公式サイト から「Evergreen Bootstrapper」をダウンロードしてインストールしてください。
  • 企業のグループポリシーで WebView2 がブロックされている場合は、IT 管理者にご相談ください。

オンライン機能が動作しない場合

OSV API や PyPI / npm / NuGet への接続がうまくいかない場合、以下を確認してください。

  • インターネット接続: ブラウザで通常のウェブサイトにアクセスできるか確認してください。
  • ファイアウォール: 企業ネットワークの場合、api.osv.dev、pypi.org、registry.npmjs.org、api.nuget.org への HTTPS 通信が許可されているか確認してください。
  • プロキシ: プロキシ環境では、Windows のシステムプロキシ設定が正しく構成されている必要があります。
  • Pro ライセンス: オンライン機能は Pro ライセンスが必要です。設定画面でライセンス状態を確認してください。

サポートへの問い合わせ

上記で解決しない場合は、以下の方法でサポートにお問い合わせください。

  • メール: support@sumikkolab.com
  • サポートページ: サポート

お問い合わせの際は、以下の情報をお知らせください。

  • Dependa のバージョン（Dependa.exe version で確認可能）
  • Windows のバージョン
  • エラーメッセージの全文（スクリーンショットがあると理想的です）
  • 再現手順

Common v2.0 Issues

SBOM File Cannot Be Imported

Check the following:

  • The file format must be CycloneDX 1.4/1.5 JSON or SPDX 2.3 JSON.
  • XML format SBOMs are not supported in v2.0. Convert to JSON format before importing.
  • The file is not corrupted. Verify it can be parsed as valid JSON.

License Preload Error

If "License preload failed" appears at startup, there is an issue loading the license database. Restart the app, or delete the settings file and restart.

Review Count Stops at 5

The Free edition limits reviews to 5 items. A Pro license is required to review all packages.

Report Shows Watermark

HTML reports generated in the Free edition include a "Free Edition" watermark. This is expected behavior. Upgrade to Pro for watermark-free reports.

Settings screen — check Pro license status and online settings when troubleshooting

Resetting Settings

To reset settings to their defaults:

  1. Close Dependa.
  2. Delete the following file:
%LocalAppData%\Dependa\settings.json

The settings file will be recreated with default values on the next launch.

Deleting the settings file also resets the terms of use acceptance. You will need to agree again.

WebView2 Runtime Not Found

Dependa uses Microsoft Edge WebView2 Runtime for parts of its UI. If you see an error about WebView2:

  • Windows 11 includes it by default. No installation is usually needed.
  • On Windows 10, download the "Evergreen Bootstrapper" from Microsoft's official site and install it.
  • If WebView2 is blocked by your organization's group policy, contact your IT administrator.

Online Features Not Working

If OSV API or PyPI / npm / NuGet connections fail, check the following:

  • Internet connection: Verify you can access regular websites in your browser.
  • Firewall: In corporate networks, ensure HTTPS access to api.osv.dev, pypi.org, registry.npmjs.org, and api.nuget.org is allowed.
  • Proxy: In proxy environments, Windows system proxy settings must be correctly configured.
  • Pro license: Online features require a Pro license. Check your license status in Settings.
